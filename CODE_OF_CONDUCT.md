# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## TL;DR

- Be respectful. Assume good faith.
- Focus on the work — code, tests, docs, the API surface.
- Disagree on technical merit, not on people.
- No personal attacks, no demographic slurs, no off-topic noise.

## Scope

Applies to issues, pull requests, discussions, code review comments, and any other
project space (including private channels when representing the project).

## Enforcement

Report concerns to **dev.ungheni@gmail.com**. The maintainer reviews privately and
responds within a reasonable window. Outcomes range from a private warning to
temporary or permanent bans from the project. Decisions are final and not subject to
appeal in public threads.
